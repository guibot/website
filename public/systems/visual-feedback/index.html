Reactive Visual Feedback System

20 November 2025 archived
visual interaction feedback

Concept

An audiovisual feedback system exploring non-linear video processing through recursive routing and real-time manipulation. The system creates emergent visual patterns from simple input sources.

Process

Developed using TouchDesigner with custom GLSL shaders for feedback operations. The system routes camera input through a chain of effects, feeding the output back into itself with controllable delays and transformations.

Technical stack:

  • TouchDesigner for signal flow
  • Custom GLSL shaders for video processing
  • MIDI controllers for parameter mapping
  • NDI output for multi-display setups

Current Status

Project archived after exhibition at [Venue Name] in 2025. Code and documentation available on request. Learnings from this system informed the development of later audiovisual tools.
