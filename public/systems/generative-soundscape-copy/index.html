Generative Soundscape Engine

15 January 2026 ongoing
sound generative real-time

Concept

A modular system for generating evolving soundscapes based on algorithmic composition and real-time parameter control. The system explores emergent behavior through layered synthesis and probabilistic sequencing.

Process

Built using Pure Data and custom C++ audio processing modules. The architecture separates sound generation from control logic, allowing for flexible routing and experimentation with different generative strategies.

Key components:

  • Multi-voice granular synthesis engine
  • Markov chain-based sequence generator
  • Spatial audio processing (8-channel output)
  • OSC-based external control interface

Current Status

Active development. Currently testing integration with motion tracking for interactive performances. Next phase involves expanding the control vocabulary and exploring machine learning for parameter evolution.
